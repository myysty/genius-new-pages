# How to install?

Get a userscript manager like [Violentmonkey](https://violentmonkey.github.io/) or [Tampermonkey](https://www.tampermonkey.net/) for your browser, follow [this raw file link](https://github.com/myysty/genius-new-pages/raw/refs/heads/master/sadasd.user.js) and complete installation as prompted.
